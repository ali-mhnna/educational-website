# Educational Website

## Laravel + React Educational Platform

### Installation

#### Backend Setup
1. cd backend
2. composer install
3. cp .env.example .env
4. php artisan key:generate
5. php artisan migrate
6. php artisan serve

#### Frontend Setup
1. cd frontend
2. npm install
3. cp .env.example .env
4. npm start

### Requirements
- PHP 8.2+
- MySQL
- Node.js 16+
- Composer
